## 🏃 Quick Start

### Prerequisites
- Python 3.x installed
- Node.js installed
- Git installed

### Setup
```bash
# Clone repository
git clone https://github.com/VelyrionHQ/velyrion-protocol.git
cd velyrion-protocol

# Backend setup
cd backend
python -m venv venv
venv\Scripts\activate     # Windows
pip install -r requirements.txt

# Frontend setup
cd ..\frontend
npm install

# Contracts setup
cd ..\contracts
npm install
```

### Run Development
```bash
# Use startup script
start-dev.bat

# Or manually:

# Terminal 1 - Backend
cd backend
venv\Scripts\activate
python main.py

# Terminal 2 - Frontend  
cd frontend
npm run dev

# Terminal 3 - Contracts
cd contracts
npx hardhat test
```
